# Trusted By logo files

Drop logo files in this folder and the /start Trusted By bar picks them up
automatically — no code changes needed. Until a file exists, that
organization renders as a styled wordmark.

Use these exact filenames (`.svg` preferred, `.png` also works — SVG is
tried first):

| Organization | Filename |
|---|---|
| Lululemon | `lululemon.svg` or `lululemon.png` |
| The Coca-Cola Company | `coca-cola.svg` / `.png` |
| Google Women Techmakers | `women-techmakers.svg` / `.png` |
| Femme Alliance Network Canada | `femme-alliance.svg` / `.png` |
| ACCESS BIO | `access-bio.svg` / `.png` |
| Hy-Initiative | `hy-initiative.svg` / `.png` |
| Springfield Tech Council | `springfield-tech-council.svg` / `.png` |
| World Mission Agency | `world-mission-agency.svg` / `.png` |
| Center for AI and Digital Policy | `caidp.svg` / `.png` |

Guidelines:
- Transparent background, roughly 3:1 or wider aspect, at least 64px tall
  (they display at 32px height, grayscale at rest, brand color on hover).
- Light/white logo variants read best on the dark site background.
- Per the PRD, confirm usage permission for each logo before adding it.
